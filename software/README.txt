Si vous voulez tester localement le site:

1. Créer un fichier ".env" dans le dossier "software"
2. Ecrire "SESSION_SECRET=secret" dedans
3. Penser à installer tous les modules NPM ainsi que les modules mis en catégorie développeur
3. Done!

NOTES : - Squelette très basique du site fait pour se faire une idée en se basant sur les fonctions API que tu m'avais donné
        - Il reste certaines fonctions API en relation avec l'ESP et le mail de confirmation à configurer
